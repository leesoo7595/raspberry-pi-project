11/5 ~ 실습예정

1. 프로젝명 : 아기관찰 카메라
2. 금주 실습할 내용
- 실습명 : 라즈베리 카메라 키트 실습
- 실습절차 <br>
가. 라즈베리 카메라와 라즈베리파이 연동 <br>
나. 스마트폰과 라즈베리파이의 연동 <br>
다. 라즈베리파이로 사진을 찍는다. <br>
라. 연동된 스마트폰을 통해 라즈베리파이에서 찍은 사진을 받아볼 수 있다. <br>
마. 추가적으로 동영상도 가능한지 확인한다. <br>
바. 이미지 파일을 핸드폰이나 노트북으로 어떻게 확인할 수 있을지 제대로 알아본다. <br>
