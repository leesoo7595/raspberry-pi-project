11/5 ~ 실습예정

1. 프로젝명 : 아기관찰 카메라
2. 금주 실습할 내용
- 실습명 : 라즈베리 카메라 키트 실습
- 실습절차 <br>
가. 라즈베리 카메라와 라즈베리파이 연동 <br>
나. 스마트폰과 라즈베리파이의 연동 <br>
다. 라즈베리파이로 사진을 찍는다. <br>
라. 연동된 스마트폰을 통해 라즈베리파이에서 찍은 사진을 받아볼 수 있다. <br>
마. 추가적으로 동영상도 가능한지 확인한다. <br>
바. 이미지 파일을 핸드폰이나 노트북으로 어떻게 확인할 수 있을지 제대로 알아본다. <br>

3. 관련 링크

## 11/12까지 실습한 내용
1. 실습명 : opencv로 얼굴인식 테스트
2. 실습 절차
- opencv-python 라이브러리 다운로드 및 import 하기
- 위 링크의 코드를 참고하여 opencv의 얼굴인식 모델 데이터를 공식 opencv 깃헙에서 가져옴
- 해당 데이터를 이용해서 얼굴 인식 테스트하기
- 백퍼센트 정확하진않지만 어느정도 성공
3. 참고 링크

## 11/19까지 실습할 내용 계획
1. 실습명 : 라즈베리파이와 Mac 원격 연결 성공 및 파이카메라 실습
2. 실습 절차
- Mac과 라즈베리파이를 원격으로 연결하여 Mac에서 접속가능하게 하기
- 라즈베리파이에 파이카메라를 연결하기
- 라즈베리파이에서 파이카메라를 이용하여 사진 및 동영상 찍어보기
- 사진 및 동영상 파일 형식을 확인하고, 이를 Mac으로 불러올수 있는지 테스트하기
- 이에 대한 결과를 실습일지로 쓰기
